# 🏀 Liga de Básquetbol

## Descripción

Este proyecto consiste en un sistema web desarrollado con Laravel para la gestión de una liga de básquetbol.

El sistema permite administrar equipos, jugadores y partidos mediante operaciones de registro, consulta, edición y eliminación de información.

## Instalación

1. Clonar el repositorio.

```bash
git clone https://github.com/OdinSama789/liga-basquetbol.git
```

2. Ingresar al proyecto.

```bash
cd liga-basquetbol
```

3. Instalar dependencias.

```bash
composer install
```

4. Copiar el archivo de configuración.

```bash
copy .env.example .env
```

5. Generar la clave de Laravel.

```bash
php artisan key:generate
```

6. Configurar la base de datos en el archivo `.env`.

7. Ejecutar las migraciones.

```bash
php artisan migrate
```

8. Iniciar el servidor.

```bash
php artisan serve
```
